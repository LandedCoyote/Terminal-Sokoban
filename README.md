# Terminal Sokoban (2D)

A simple 2D Sokoban game designed to run on a terminal.

## Controls
- **Arrow Keys**: Move Up/Down/Left/Right
- **Z Key**: Confirm / Action
- **X Key**: Cancel / Back

# Terminal Sokoban (2D)

ターミナル上で動作する2D倉庫番ゲームです。

## 操作方法
- **矢印キー**: 上下左右の移動
- **Zキー**: 決定 / アクション
- **Xキー**: キャンセル / 戻る
